## 📖 About

This project uses prompt engineering to generate
professional website copy for local businesses.
It covers 3 business types — café, salon, and clinic —
with Homepage, Services, and CTA sections for each.

4. Review output against the quality checklist
5. Copy into your CMS, Framer, or Webflow
